# benchmark-analysis

A Python project for benchmarking and analyzing the performance of common sorting and searching algorithms. It measures execution times across varying input sizes, then generates visualizations to compare algorithm efficiency.

## Setup

1. Make sure Python is installed on your machine.
2. Create a project folder called `benchmark-analysis`.
3. Inside the folder, create a virtual environment:
   ```
   python -m venv venv
   ```
4. Activate it:
   - Windows: `venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
5. Install the one dependency:
   ```
   pip install matplotlib
   ```
6. Create three files: `algorithms.py`, `benchmark.py`, `analysis.py`
